/// TO USE STRUCTCHECK: ///
Place the following files in the folder containing the packages/classes:
	- checkstyle-X.X.X-all.jar (replace X's with numbers corresponding to the jar's version)
	- config.xml
	- StructCheck.jar
	- StructCheck_Exec.py
In Eclipse IDE, the checkstyle and StructCheck jar files must be referenced in the Java project.
	
To execute, use command "python StructCheck_Exec.py" or "python3 StructCheck_Exec.py"
in the Windows CMD or Linux Terminal while in the folder containing the file.
